# Turf Booking System – Web Interface

This project implements the web interface of a Turf Booking System using HTML and CSS.

The system allows users to browse available sports turfs, check booking slots, and reserve a turf online. 
It also includes a simple admin dashboard to manage bookings and turf information.

## Pages Included
• Home Page  
• Turf Listing Page  
• Booking Form  
• Login Page  
• Admin Dashboard

HTML  
CSS
